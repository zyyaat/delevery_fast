Error Budget Policy / سياسة ميزانية الأخطاء
If a service burns >50% of its monthly error budget in any 7-day window → deployment freeze for non-critical changes to that service. Focus shifts to reliability work. Budget resets monthly. Tier 0 violations trigger automatic incident review.

TECHNICAL BLUEPRINT v1.0
- 3-tier SLO framework: Tier 0 critical (99.95%), Tier 1 important (99.9%), Tier 2 standard (99.5%). - Tier 0 services: Order, Payment, Auth — 22 min/month error budget. - 4 Golden Signals: Latency, Traffic, Errors, Saturation — Google SRE classic. - Error budget policy gates deployments — >50% burn in 7d = freeze. - Tier 0 violations trigger automatic incident review.
