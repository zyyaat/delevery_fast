listIndexes / الفهارس

  • PK PRIMARY KEY (id)
  • BTREE (user_id, created_at DESC) — customer order history
  • BTREE (restaurant_id, status) — restaurant incoming orders
  • BTREE (driver_id, status) — driver active deliveries
  • BTREE (status, created_at) — ops dashboard filtering
  • GIST (lat, lng) — PostGIS spatial queries

call_splitPartitioning / التقسيم

Range partitioning by created_at month. Each partition holds ~1.5M rows at peak. Old partitions (>12 months) moved to cold storage (S3 + Parquet). Active partition kept on hot NVMe storage; query planner prunes partitions automatically.

verifiedConstraints / القيود

  • CHECK (total_amount >= 0)
  • CHECK (delivered_at IS NULL OR delivered_at > created_at)
  • CHECK (status IN valid enum values)
  • FK to users, restaurants, drivers (with ON DELETE RESTRICT)
  • UNIQUE (payment_provider_txn_id) where not null
