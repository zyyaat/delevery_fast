Promotions & Loyalty Engine / محرك العروض والولاء

Coupons, campaigns, points, cashback — all configurable without code deploys.

local_offer

Coupon Types

أنواع الكوبونات
  • Flat discount — EGP 20 off orders >EGP 100
  • Percentage — 15% off, capped at EGP 50
  • Free delivery — no fee
  • BOGO — buy one get one (menu_item scoped)
  • First-order bonus — new users only
Each coupon has: code, valid window, usage limit per user, total usage limit, minimum order value, applicable restaurants/items.
campaign

Campaign Targeting

استهداف الحملات
  • Segment-based: new users, dormant users (no order in 30 days), high-value (top 10% GMV)
  • Geo-based: per city, per district
  • Time-based: lunch hours, weekday vs weekend
  • Behavior-based: viewed restaurant X but didn't order
Campaigns are evaluated in real-time when the Customer App loads /home — returning personalized banners.
stars

Loyalty Points

نقاط الولاء
  • 1 point per EGP 1 spent (excl. delivery fee)
  • 2× points during birthdays
  • 100 points = EGP 5 discount
  • Tier system unlocks perks:
    SILVER · 1K pts GOLD · 5K pts PLATINUM · 15K pts
  • Tiers unlock: priority support, free delivery, exclusive restaurant access
Points expire 12 months after earning — encourages redemption.
shield

Anti-Abuse Rules

قواعد منع الإساءة
  • Max 1 coupon per order
  • Max 3 coupons per user per month
  • New device + new number + same IP → flagged for review
  • Coupon flooding detected via velocity rules (5 redemptions in 1hr from same area = block)
  • Manual review queue for high-value coupons (>EGP 50 discount)
Velocity rules + device fingerprinting = layered defense against promo abuse.
